Design Elements

Key visual decisions in this hybrid

🎨 Color Palette

Deep navy background (#1a1a2e), muted text (#c8c8d4), soft purple accent (#6c6ce0), green for healthy (#5a9e6f), amber for warnings (#d4a04e), red for errors (#c05050)

📐 Borders & Layout

Thin left-accent on selected row, subtle separators, rounded corners on info bars. No heavy box borders — whitespace defines sections.

✨ Icons

Unicode icons for context: 📁 repo, 🤖 agent, 📝 editor, 🌿 branch, 💻 tmux, 🌳 worktree. Status dots: ● active, ○ idle, ◐ in-progress.

🏷️ View Headers

Each view gets a distinctive icon: ◈ list, ✦ create, ⚠ cleanup confirm. Keeps navigation clear without heavy chrome.
